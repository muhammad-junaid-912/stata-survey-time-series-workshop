# STATA Survey and Time Series Analysis Workshop

This repository documents a hands-on training workshop on survey data analysis and time series analysis using STATA.

##  Workshop Details
- Title: Stata for Survey Data Analysis and Time Series Data

##  Objectives
- Data cleaning and preparation
- Descriptive statistics
- Survey analysis techniques
- Time series modeling
- Reproducible research using do-files

##  Repository Structure
- data/ → Raw datasets used in analysis  
- do-files/ → STATA scripts  
- outputs/ → Tables and results  
  

##  Software Used
- STATA
- Excel (for export)
- GitHub for version control
